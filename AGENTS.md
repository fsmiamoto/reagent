## Build/Test Commands
- `bun run build` - Build both server and UI
- `bun run build:server` - Build TypeScript server only
- `bun run build:ui` - Build React UI only
- `bun test` - Run all tests
- `bun run lint` - Run linting for project
- `bun run lint:fix` - Apply project formatting style
- `bun run test:watch` - Run tests in watch mode
- `vitest run tests/path/to/file.test.ts` - Run single test file

## Code Style
Follow existing patterns in the codebase. Prefer clarity over cleverness.
